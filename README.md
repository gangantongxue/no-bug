# 🧘‍♂️ nb - 佛祖保佑，no bug！

> 一个能让你的代码充满佛光的神奇工具

## ✨ 功能介绍

nb（读作"no bug"）是一个用Go语言编写的命令行工具，它能在你的代码文件开头**以注释的形式插入一座佛像**，并虔诚地写下"no bug"的祝福语。

有了它，你的代码将被佛祖保佑，从此远离bug的困扰！

## 📚 支持的语言

nb工具支持几乎所有主流编程语言，包括但不限于：

- Go、JavaScript、TypeScript、Java、C/C++
- Python、Ruby、PHP、Shell脚本、Perl
- Swift、Kotlin、Rust、Scala、Dart
- HTML、CSS、XML、Vue、SQL
- 以及更多...（只要是有注释的语言，它都能搞定）

## 🚀 安装方法

### 使用 go install 安装（推荐）

```bash
go install github.com/gangantongxue/no-bug/cmd/nb@latest
```

### 从源码编译

```bash
git clone https://github.com/gangantongxue/no-bug.git
cd no-bug
go install ./cmd/nb
```

## 💻 使用方法

使用方法非常简单，只需在命令行中输入：

```bash
# 处理单个文件
nb your-file.go

# 处理多个文件
nb file1.js file2.py file3.css
```

## 📝 注意事项

1. 工具会直接修改原文件，请确保你的文件已经备份
2. 支持的文件类型由扩展名决定，如果你的文件没有扩展名，可能无法正确处理
3. 如果你是无神论者，可能效果会打折扣
